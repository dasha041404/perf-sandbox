# Perf Sandbox

Сравнение производительности различных шаблонизаторов

## Цель

Разработать песочницу для сравнения производительности различных шаблонизаторов с возможностью сохранения и просмотра полученных метрик, а также построения heat-map на их основе.

## Команда 

Состоит из 5 человек которые занимаются проектированием веб-интерфейса песочницы и размещением результатов измерения во внешней базе данных

## Функциональные требования

### План минимум

- Окно для ввода строки шаблона (без валидации)
- Выпадающий список выбора шаблонизатора
- Кнопка запуска эксперимента (компиляция + замер времени)
- Сохранение результата в БД
- Страница с таблицей результатов
- Выбор строк для построения heat-map
- Heat-map: Y — шаблонизаторы, X — выходная строка, цвет — время компиляции

### План максимум

- Выбор дополнительных шаблонизаторов (мультиселект)
- Микро-LLM на бэке для трансляции шаблона в нужный синтаксис
- Модальные окна: выбор → загрузка → верификация
- Docker-образы для фронта и бэка

## Структура проекта

Структура проекта будет дополняться

## API

- `POST /experiment` — сохранить результат эксперимента
- `GET /experiments` — получить список экспериментов
- `GET /translate_template` — трансляция через LLM (максимум)
