Подключение figma

Добавить в mcp.json 
{
    "inputs": [],
    "servers": {
        "figma": {
            "url": "https://mcp.figma.com/mcp",
            "type": "http"
        }
    }
}

После сохранения должен появиться Start над сервером figma; нажми его и пройди авторизацию через свой Figma-аккаунт.

При отправке запроса необходимо вставлять ссылку на figma frame, не весь макет а отдельыне его части
